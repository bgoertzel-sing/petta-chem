# exp02 — random polymer controls

First tiny deterministic exp02 sweep.  The PeTTa kernel defines a seed-derived random-polymer rule set and two controls (shuffled catalysts and no catalysis), scans all rule pairs with the conservative exp01 ACS scanner, and records each run in the v0.1 run-contract atom shape.

This is a plumbing/control smoke test, not an emergence claim.

```bash
scripts/run_exp02.sh
```
